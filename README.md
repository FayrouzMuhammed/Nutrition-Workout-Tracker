# Nutrition-Workout-Tracker
Nutrition &amp; Workout Tracker combines technology and health awareness to create a practical personal wellness tool. This project supports Sustainable Development Goal 3 (Good Health and Well-being) by empowering users to take control of their nutrition and fitness in an organized and efficient way.

## Overview
The **Nutrition & Workout Tracker** is a health-focused application designed to help users monitor and improve their daily lifestyle by tracking meals, calorie intake, workouts, and overall fitness progress.

##  Features
•	**Daily Meal Tracking** – Log food intake and monitor calories consumed  
•	**Workout Tracking** – Record exercises and estimate calories burned  
•	**Calorie Balance System** – Compare calories consumed vs burned  
•	**Progress Monitoring** – Track daily and weekly health progress  
•	**OOP-Based Design** – Clean structure using classes and objects  
•	**Daily Consistency Tracking** – Helps users build healthy habits  
•	**Health Awareness Support** – Encourages better lifestyle decisions  
•	**Lightweight & Simple UI/Logic** – Easy to use for all users  

---
## Core Features
**1- User Profile Setup**  
The user logs in and creates a personal profile by entering:  
o	Name  
o	Age  
o	Gender  
o	Weight  
o	Height  
o	Activity days per week  

**2- The user sets personal goals:**  
o	Target weight  
o	Duration (in weeks) to reach the target weight  

**3- Meal Tracking**  
o	The user selects meals from the available food list  
o	Select food items from the food list  
o	Enter the quantity/amount for each item  
o	The system automatically calculates and records calorie intake    

**4- Viewing Meal Details**  
o	The system displays full nutritional information for the selected meal 

**5- Workout Tracking**  
o	The system calculates and updates:  
o	Total calories consumed  
o	Total calories burned  

**6- Saving and Managing Logs**  
o The user clicks “Save Logs” to store daily records  
The user can also:  
o	Clear ALL saved data using “Clear Logs”
